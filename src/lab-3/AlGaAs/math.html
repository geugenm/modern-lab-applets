Математический анализ

Минимумы долин зоны проводимости в [эВ] при 300K:    AlxGa1-xAs                    [1]

    Eg(x) =     1.424 + 1.247x                              (0 < x < 0.45)
                     1.424 + 1.247 + 1.147(x-0.45)2     (0.45 < x < 1.0)

    EL(x) = 1.708 + 0.642x

    EX(x) = 1.900 + 0.125 + 0.143x2
 

E(k)  Диаграмма: эффективные массы для  AlxGa1-xAs   [1,  2]

    A) Долины зоны проводимости
              Eg(k) = h2k2/2mg*   где,
                        mg*/m0 = 0.067 + 0.083x (для плотности состояний); 0.067 + 0.083x (для проводимости)
                        kg = 2p/a (0, 0, 0)

             EL(k) = h2(k-kL)2/2mL*   где,
                        mL*/m0 = 0.56 + 0.1x (для DOS); 0.11 + 0.03x (для поводимости)
                        kL = 2p/a (1/2, 1/2, 1/2)
 

             EX(k) = h2(k-kX)2/2mX*   где,
                        mX*/m0 = 0.85 - 0.14x (для DOS); 0.32 - 0.06x (для проводимости)
                        kX = 2p/a (0, 0, 1-D)
 

    B) Долинны валентной зоны
             Ehh(k) = h2k2/2mhh*   где,
                       mhh*/m0 = 0.62 + 0.14x

             Elh(k) = h2k2/2mlh*   где,
                       mlh*/m0 = 0.087 + 0.063x

                 Eso(k) = h2k2/2mso*  - Do    где,
                       mso*/m0 = 0.15 + 0.09x
                     Do = 0.34 - 0.04x

    Плотность состояний еффективной массы для валентной зоны, mvb, находится из
                mvb = (mlh*3/2 + mhh*3/2)2/3
    Eso(k)  является обособленной зоной и не вносит вклад в DOS.
